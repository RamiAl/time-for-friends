# Time For Friends

"Time For Friends" is a web application that allows you to track your friends local time live.

#### Tech stack:  
- MongoDB 
- Node.js
- Express.js 
- GraphQL 
- React.js


## To get started 

Install necessary packages:
```
npm install
```

## Backend

### Running frontend:
To run the backend without nodemon:
cd backend
```
node index
```

wWith nodemon:
cd backend
```
nodemon index
```

## Frontend

### Running frontend:

Start development server:  
```
npm start
```
